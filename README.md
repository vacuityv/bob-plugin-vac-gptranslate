## 使用教程

1. 安装 [Bob](https://bobtranslate.com/guide/#%E5%AE%89%E8%A3%85)，一款 macOS 平台的翻译和 OCR 软件；
2. 下载此插件: [bob-plugin-vac-gptranslate.bobplugin](https://github.com/vacuityv/bob-plugin-vac-gptranslate/releases)
3. 安装此插件: 以上两步完成之后，双击 `bob-plugin-vac-gptranslate.bobplugin` 文件即可安装；
4. 在Bob的设置中，选择GPTranslate作为翻译插件；

<img src="https://github.com/vacuityv/bob-plugin-vac-gptranslate/blob/main/img/add.jpg" width="800px">

5. 填写注册邮箱和密码

注：服务器可以选择香港，如香港无法访问可选择美国

## 获取注册邮箱和密码

1. 访问 [此网站](https://chat.vacuity.me/)，注册账号；

<img src="https://github.com/vacuityv/bob-plugin-vac-gptranslate/blob/main/img/site.jpg" width="800px">


2. 注册成功后即可使用填写注册邮箱和密码登录bob插件
3. 新注册用户系统会赠送5的余额（如果你只是用来做翻译这个余额可以使用超级久）
4. 你可以随时点击网站左侧“消耗”菜单查看账户的消耗信息。
